# Proof API Surface

HTTP API endpoints for the CROWN proof system. All endpoints require authentication via API key or session token.

| Method | Path | Description |
|---|---|---|
| POST | `/v1/proof/jobs` | Create a new proof job |
| GET | `/v1/proof/jobs/:proofJobId` | Get proof job status |
| GET | `/v1/proof/jobs/:proofJobId/chunks` | Get proof chunks (paginated) |
| GET | `/v1/proof/answers/:answerId/receipt` | Get CROWN receipt for an answer |
| GET | `/v1/proof/receipts/:receiptId/proofpack` | Get complete proof pack |
| GET | `/v1/receipts/:answerId/verify-chain` | Verify receipt chain integrity |

### POST `/v1/proof/jobs`

Create a new proof job

**Request body:**

```json
{ tenantId, agentId, artefact_id, mode, idempotencyKey?, metadata? }
```

**Response:**

`{ id, status, createdAt }`

### GET `/v1/proof/jobs/:proofJobId`

Get proof job status

**Response:**

`{ id, status, progress, chunks?, createdAt, completedAt? }`

### GET `/v1/proof/jobs/:proofJobId/chunks`

Get proof chunks (paginated)

**Response:**

`{ chunks: [...], cursor?, hasMore }`

### GET `/v1/proof/answers/:answerId/receipt`

Get CROWN receipt for an answer

**Response:**

`CrownReceipt (see JSON Schema)`

### GET `/v1/proof/receipts/:receiptId/proofpack`

Get complete proof pack

**Response:**

`{ receipt, evidence, chain, integrity }`

### GET `/v1/receipts/:answerId/verify-chain`

Verify receipt chain integrity

**Response:**

`{ valid, depth, receipts, latencyMs }`
