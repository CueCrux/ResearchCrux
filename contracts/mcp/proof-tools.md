# Proof MCP Tools

Detailed documentation for the Proof domain MCP tools. These tools manage the proof lifecycle: submitting documents for proof, checking status, retrieving receipts and proof packs.

## `proof_document`

Submit a document for proof generation. Creates a proof job that processes the document, generates CROWN receipts for each answer chunk.

**Parameters:**

`artefact_id` (required), `mode` (`light`|`verified`|`audit`), `metadata` (optional object)

**API Endpoint:** `POST /v1/proof/jobs`

## `get_proof_status`

Check the status of a proof job. Returns current state (queued, processing, completed, failed) and progress.

**Parameters:**

`proof_job_id` (required)

**API Endpoint:** `GET /v1/proof/jobs/:proofJobId`

## `get_proof_chunks`

Retrieve processed chunks from a completed proof job. Supports cursor-based pagination.

**Parameters:**

`proof_job_id` (required), `cursor` (optional)

**API Endpoint:** `GET /v1/proof/jobs/:proofJobId/chunks`

## `get_proof_receipt`

Retrieve the CROWN receipt for a specific answer. Returns the full receipt with hash, signature, and evidence.

**Parameters:**

`answer_id` (required)

**API Endpoint:** `GET /v1/proof/answers/:answerId/receipt`

## `get_proofpack`

Retrieve a complete proof pack for a receipt. Bundles the receipt, evidence records, and chain metadata.

**Parameters:**

`receipt_id` (required)

**API Endpoint:** `GET /v1/proof/receipts/:receiptId/proofpack`
